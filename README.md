## Style

This site uses the shared **Oranburg style system** (`assets/css/oranburg-style.css`), which is the canonical design foundation across Professor Oranburg's academic sites. It features:

- **Dark mode default** with light mode toggle (persisted in `localStorage`)
- **Color palette:** deep navy (`#0A3255`), crimson red (`#B21F2C`), sky blue (`#6DACDE`), gold (`#FFD65C`), teal (`#B5E1E1`)
- **Typography:** Oswald (headlines), Crimson Text (body/accent), Roboto (UI)
- **Responsive** mobile-first layout
